# StudentApp - Student Management Portal

This React application demonstrates component composition using `Home`, `About`, and `Contact` components as part of the **Week 5 React Components Hands-On Lab**.

## 🚀 Components Included

- **`Home`**: Displays *"Welcome to the Home page of Student Management Portal"*
- **`About`**: Displays *"Welcome to the About page of the Student Management Portal"*
- **`Contact`**: Displays *"Welcome to the Contact page of the Student Management Portal"*

## 🛠️ Execution

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm start
   ```

3. Open [http://localhost:3000](http://localhost:3000) to view the portal.
