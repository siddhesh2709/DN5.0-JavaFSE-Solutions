# myfirstreact Application

This project is created as part of the **Week 5 React & SPA Hands-On Lab**.

## 🚀 Getting Started

### Prerequisites
- Node.js (v14+)
- NPM

### Installation
Install project dependencies:
```bash
npm install
```

### Running the Application
Start the development server:
```bash
npm start
```
Open [http://localhost:3000](http://localhost:3000) to view the page rendering:
> **welcome to the first session of React**
